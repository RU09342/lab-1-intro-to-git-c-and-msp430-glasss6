## Function
### math(int num1, int num2, char Operator)
Argument 1: Integer
Argument 2: Integer
Operation: Operation to performed on two integers
* '+' add two integers
* '-' subtract two integers
* '/' divide two integers
* '%' modulus two integers
Return value: Integer
